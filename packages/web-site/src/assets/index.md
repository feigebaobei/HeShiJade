# 为什么叫和氏璧

我曾在公司两次开发低代码项目，都失败了。心理知道什么才是好的低代码。这些在公司里都不能实现了。我只好自己实现。这次不再为公司做了，为整个it界做。我三次做低代码与卞和三次献宝和氏璧相呼应，所以把第三次做的低代码叫做和氏璧。
