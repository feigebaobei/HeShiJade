增加组件

搭建侧增加组件

  1. 在`constructor/src/helper/category.ts`的组件列表中增加组件项
  2. 在`constructor/src/helper/props.ts``constructor/src/helper/behavior.ts``constructor/src/helper/items.ts``constructor/src/helper/gridLayout.ts`设置组件的配置项props/behavior/items/gridLayout
  3. 在`constructor/src/helper/addable.ts`中新组件的items是否可增加
  4. 在`constructor/src/helper/components.ts`中引入no.2中创建的props/behavior/items设置该组件的默认值
  5. 在`constructor/src/app/props/props-box/props-box.component.ts`/`constructor/src/app/behavior/behavior-box/behavior-box.component.ts`中分别引入并处理props、behavior
  6. 初始化组件。在constructor目录下执行`ng g c components/new-comp`
  7. 独立组件
    • 注释`constructor/src/app/components/new-comp/new-comp.component.ts`中的引入相关依赖的module，再选择基于InputData或TextBase或ListenItems开发。
    非独立组件
    • 注释`constructor/src/app/components/new-comp/new-comp.component.ts`中的`standalone``imports`，再选择基于InputData或TextBase或ListenItems开发。
    • 在`constructor/src/app/components/components.modules.ts`中引入新组件并声明。如果新组件使用了特定的module，则需要在此文件中引入并使用。
  8. 在`constructor/src/app/components/comp-box/comp-box.component.ts`中引入新组件并传入数据
  9. 在新组件中接收数据、处理逻辑、监听事件、ngOnInit。（可以参考flex组件）

渲染侧增加组件

  1. 在`renderer`目录下执行`ng g c components/new-comp`
  2. 独立组件
    • 在`renderer/src/app/components/new-comp/new-comp.components.ts`中的引入相关依赖的module。
    非独立组件
    • 注释`renderer/src/app/components/new-comp/new-comp.components.ts`中的`standalone``imports`。
    • 在`renderer/src/app/components/components.module.ts`引入新组件并声明。如果新组件使用了特定的module，则需要在此文件中引入并使用。
  3. 在`renderer/src/app/components/comp-box/comp-box.component.ts`中引入新组件并为其传入数据
  4. 在`renderer/src/app/components/new-comp/new-comp.components.ts`中引入CompBase类，再基于它开发组件。可以参考Radio组件。
  5. 参考搭建侧的同组件，编写渲染侧的 *.html/*.css

官网侧增加组件

  1. 在`HeShiJade/packages/web-site/src/app/api/api.component.html`中增加新组件的api.包含props/slots/method/event/items.

增加配置项

  1. 在`constructor`目录下执行`ng g c props/props-xxx`
  2. 在`constructor/src/app/props/props-item.component.ts`中引入新组件、传入数据、绑定事件
  3. 在`constructor/src/app/props/props-xxx.component.ts`中引入相关依赖module。接收data、change。定义modelChangeH方法。
  4. 在`constructor/src/app/props/props.module.ts`中引入新组件并声明。
  5. 在新组件中处理传进来的数据、触发事件、去抖处理、调用接口。

增加导航（待升级）

  1. 在`web-site`目录下执行`ng g c xxx`
  2. 在`web-site/src/app/app-routing.module.ts`中引入新组件并绑定路由。
  3. 在`web-site/src/app/app.component.html`中开辟新导航并绑定事件。
  4. 在`web-site/src/app/app.component.ts`中，把no.3的回调方法中写跳转逻辑。
