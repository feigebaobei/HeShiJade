api总纲

此api作用于低代码范围。零代码用配置就能完成，不需要写代码。和氏璧（此低代码平台）基于dev-ui组件库搞的。封装了常用的组件。对外暴露4种api（props/slots/method/event）。此平台的低代码能力在事件面板上体现。

props: 组件的属性。

slots: 组件的插槽。根据实际需求，或没有，或一个，或多个。

method: 组件的方法。

event: 组件本身的事件和生命周期事件。

事件的回调方法

选中组件后在“behavior”面板选择事件，设置方法体。平台已经为回调方法设置三个形式参数（utils：内置工具集合，plugins：当前应用的选中的插件集合，thirdParams：当前事件的相关入参）

demo

创建一个loop组件，内部再创建一个input组件。在loop组件的behavior面板中添加一项，并设置为“渲染视图后”，设置方法体为下面的代码。

        let arr = [
            {value: 'a',},
            {value: 'b',},
            {value: 'c',},
        ]
        utils.pool.getComponentInstance('01KEX37KETPC58ZW38XFC653S1').setLoopValue(arr)

渲染效果是3个input组件的值分别是'a'/'b'/'c'

下文中使用的数据类型

        type S = string
        type N = number
        type A = any
        type B = boolean
        type ULID = S
        type D = Date
        type F = Function
        type O = object
        interface Oa {
            [k: S]: A
        }
