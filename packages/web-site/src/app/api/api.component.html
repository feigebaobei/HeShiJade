api总纲

此api作用于低代码范围。零代码用配置就能完成，不需要写代码。和氏壁（此低代码平台）基于dev-ui组件库搞的。封装了常用的组件。对外暴露4种api（props/slots/method/event）。此平台的低代码能力在事件面板上体现。

props: 组件的属性。

slots: 组件的插槽。根据实际需求，或没有，或一个，或多个。

method: 组件的方法。

event: 组件本身的事件和生命周期事件。
